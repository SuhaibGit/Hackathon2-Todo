# Form With Validation Generator Skill

## Purpose
Create React forms with comprehensive validation, error handling, and user feedback using TypeScript and Tailwind CSS.

## Function
- Build form components with controlled inputs and state management
- Implement client-side validation with immediate feedback
- Create error message display and form submission handling
- Add loading states and success feedback
- Integrate with API client methods for data submission
- Implement accessibility features for keyboard navigation

## Form Components
- Controlled input components (text, textarea, select, checkbox, radio)
- Form state management with useState/useReducer
- Validation logic with custom or library functions
- Error message display and field highlighting
- Loading indicators for submission
- Success/error feedback after submission

## Validation Features
- Real-time validation on blur or change events
- Required field validation
- Format validation (email, phone, password strength)
- Custom validation rules for business logic
- Error message localization and user-friendly text
- Submission validation before API calls

## Output Format
- Complete React form component with TypeScript typing
- Validation logic and error handling
- Tailwind CSS classes for styling
- Integration with API client methods
- Accessibility attributes and keyboard navigation
- Ready-to-use form with comprehensive validation
