# Tailwind Component Builder Skill

## Purpose
Create reusable React components with Tailwind CSS styling following design system principles and responsive patterns.

## Function
- Build modular, reusable React components with TypeScript
- Implement responsive design with Tailwind's breakpoint system
- Create accessible components with proper ARIA attributes
- Establish consistent design patterns and spacing
- Optimize for performance with proper memoization when needed
- Include component variants and state management

## Component Structure
- TypeScript interfaces for props and state
- Conditional class application based on props
- Responsive utility classes for different screen sizes
- Component composition patterns for flexibility
- Accessibility attributes and keyboard navigation
- State management for interactive components

## Design System Elements
- Consistent color palette application
- Typography scales and font weight consistency
- Spacing system using Tailwind's spacing scale
- Interactive element styling (hover, focus, active states)
- Component variants (primary, secondary, disabled, etc.)
- Responsive behavior across device sizes

## Output Format
- Complete React component with TypeScript typing
- Tailwind CSS classes for styling and responsiveness
- Prop interfaces for component customization
- Accessibility attributes and ARIA labels
- Ready-to-use component that fits design system guidelines
