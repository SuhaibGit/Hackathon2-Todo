# Next.js Skills

This document contains skill prompts for Next.js development tasks that can be used with Claude Code.

## Frontend Component Development

**Skill**: Create Next.js React components using TypeScript and Tailwind CSS
- Use functional components with TypeScript interfaces
- Implement responsive design with Tailwind CSS classes
- Follow Next.js best practices for component structure
- Include proper prop typing and error handling
- Use modern React patterns (hooks, context, etc.)

## Page Route Creation

**Skill**: Create Next.js App Router pages and routes
- Generate proper route files in the `app/` directory
- Implement dynamic routes using bracket notation `[param]`
- Use loading and error boundaries appropriately
- Include metadata for SEO when needed
- Follow Next.js conventions for route organization

## API Route Development

**Skill**: Build Next.js API routes for backend functionality
- Create API endpoints in `app/api/` routes
- Handle different HTTP methods properly
- Implement proper request/response types
- Add error handling and validation
- Follow RESTful API design principles

## Data Fetching Patterns

**Skill**: Implement Next.js data fetching strategies
- Use server-side rendering (SSR) with `async` page components
- Implement static site generation (SSG) with `generateStaticParams`
- Apply client-side fetching with `fetch` and SWR/react-query
- Handle loading states and error boundaries
- Optimize performance with caching headers

## Form Handling and Validation

**Skill**: Create forms with validation in Next.js applications
- Use controlled components with React state
- Implement form validation with libraries like Zod or Joi
- Handle form submission with proper error handling
- Integrate with API routes for data persistence
- Provide user feedback for successful submissions

## Authentication Integration

**Skill**: Implement authentication in Next.js applications
- Set up authentication providers (Auth.js, Clerk, etc.)
- Create protected routes and middleware
- Handle session management properly
- Implement login/logout functionality
- Secure API routes with authentication checks

## Styling and Theming

**Skill**: Apply consistent styling and theming in Next.js
- Configure Tailwind CSS with custom themes
- Create reusable styled components
- Implement dark/light mode support
- Maintain consistent design system
- Optimize for accessibility standards

## Performance Optimization

**Skill**: Optimize Next.js application performance
- Implement code splitting with dynamic imports
- Use Next.js Image component for optimization
- Apply lazy loading for components
- Optimize fonts and CSS loading
- Implement proper caching strategies

## Testing Practices

**Skill**: Write tests for Next.js applications
- Create unit tests for React components
- Implement integration tests for API routes
- Use testing libraries like Jest and React Testing Library
- Test user interactions and state changes
- Mock external dependencies appropriately
